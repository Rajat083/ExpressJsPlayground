# Backend With ExpressJs

This repository is meant for educational purposes, to learn how to create a backend with ExpressJs. It contains a simple API that allows you to perform CRUD operations on a list of users.

This project is structured in a way that allows you to easily understand how to set up an ExpressJs server, define routes, and handle requests and responses.

This will lso include Database connection, middlewares, authentication,  authorization, controllers, services, and more.
